## Скрипт для установки окружения и некоторых базовых программ для сетапа CachyOS+Niri

### Замечания перед установкой самой системы:

- Если вы хотите получить возможность использовать снапшоты, вы выберете BTRFS как файловую систему, однако использование её вместе с GRUB не совсем желательно (он не заработает "из коробки, нужны дополнительные настройки). В этом случае предпочтительно посмотреть в сторону Limine.
- Limine -- более современный загрузчик, и как всё "более современное" относится к размеру хранилища менее щепетильно, чем GRUB. Помните, Limine хранит в /boot/efi целиком ядра системы. Не только маленький initramfs, а прям целиком. Поэтому, если выберете его, проследите, чтобы ваш загрузочный раздел был не менее 1Gb.
- Если не хотите заморачиваться, рекомендую рассмотреть ext4. С ним GRUB будет работать без проблем
- При установке выберите в списке графических окружений Niri.

### После установки

Нажимайте Win+Enter, откроется консоль. Чтобы использовать этот скрипт, установите git и стяните себе этот репозиторий:

```bash
sudo pacman -Syu git
git clone --depth=1 ...адрес этого репозитория...
```

Вы также можете скопировать его на носитель и, так или иначе, переходим в папку с этим репозиторием и запускаем скрипт:

```bash
cd pimp-niricachyos
./init.sh
```

Скрипт установит noctalia v5 и настроит niri для работы с ней. После завершения работы скрипта нужно нажать Alt+Ctrl+Del - это команда завершения сеанса Niri. После того, как снова загрузите Niri, начав новый сеанс, вы увидите обновлённое окружение. Для вызова меню Пуск нажмите Win+Ctrl+Enter, для вызова меню сессии - Win+Shift+Q, чтобы посмотреть другие сочетания клавиш - Win+Shift+Esc. Yazi вызывается по Win+E, Neovide по Win+N, браузер по Win+B

#### Если у вас GRUB:

Чтобы добавить Windows в dual-boot в GRUB сначала проверьте, что настройка `GRUB_DISABLE_OS_PROBER=false` в `/etc/default/grub` и запускайте `sudo grub-mkconfig -o /boot/grub/grub.cfg`

### Зачем вообще это нужно?

Из коробки CachyOS c Niri cовершенно не раскрывает потенциал системы, мне захотелось сделать решение, которое бы ощущалось лишь чуть менее проработанным, чем KDE, но с гиковским духом. Поэтому в итоговом результате мы должны получить неплохо настроенный менеджер Yazi, Neovide, Noctalia. А в настройки Niri также добавлен сетап, который раскрывает её возможности по перемещению окон. В качестве просмотрщика картинок используется IMV, для чтения pdf/epub/fb2/djvu используется Zathura, архивы можно создавать в Yazi с помощью шорткатов, а просматривать в том же Yazi, для чего используется специальный скрипт. Позже я подробно опишу кастомные шорткаты для различных задач, за счёт которых эта сборка и заставляет систему играть новыми красками.
