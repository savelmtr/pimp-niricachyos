## Скрипт для установки окружения и некоторых базовых программ для сетапа CachyOS+Niri

### Замечания перед установкой самой системы:

- Если вы хотите получить возможность использовать снапшоты, вы выберете BTRFS как файловую систему, однако использование её вместе с GRUB не совсем желательно (он не заработает "из коробки", нужны дополнительные настройки). В этом случае предпочтительно посмотреть в сторону Limine.
- Limine -- более современный загрузчик, и как всё "более современное" относится к размеру хранилища менее щепетильно, чем GRUB. Помните, Limine хранит в /boot/efi целиком ядра системы. Не только маленький initramfs, а прям целиком. Поэтому, если выберете его, проследите, чтобы ваш загрузочный раздел был не менее 1Gb.
- Если не хотите заморачиваться, рекомендую рассмотреть ext4. С ним GRUB будет работать без проблем
- При установке выберите в списке графических окружений Niri.

### После установки

Нажимайте Win+Enter, откроется консоль. Чтобы использовать этот скрипт, установите git и стяните себе этот репозиторий:

```bash
sudo pacman -Syu git
git clone --depth=1 https://github.com/savelmtr/pimp-niricachyos
```

Вы также можете скопировать его на носитель и, так или иначе, переходим в папку с этим репозиторием и запускаем скрипт:

```bash
cd pimp-niricachyos
./init.sh
```

Скрипт установит noctalia v5 и настроит niri для работы с ней. После завершения работы скрипта нужно нажать Alt+Ctrl+Del - это команда завершения сеанса Niri. После того, как снова загрузите Niri, начав новый сеанс, вы увидите обновлённое окружение. Для вызова меню Пуск нажмите Win+Ctrl+Enter, для вызова меню сессии - Win+Shift+Q, чтобы посмотреть другие сочетания клавиш - Win+Shift+Esc. Yazi вызывается по Win+E, Neovide по Win+N, браузер по Win+B

#### Если у вас GRUB:

Чтобы добавить Windows в dual-boot в GRUB сначала проверьте, что настройка `GRUB_DISABLE_OS_PROBER=false` в `/etc/default/grub` и запускайте `sudo grub-mkconfig -o /boot/grub/grub.cfg`

### Шорткаты

Указываю лишь нестандартные шорткаты, которые добавил в сборку

#### Niri

- Win+N - Neovide

#### Yazi

- W - контекстное меню работы с GVFS (монтирование (m), размонтирование (u), добавление источников (a), изменение (e) и удаление источников (r))
- gm - смонтировать (если требуется) и перейти в gvfs-папку (с выбором источника)
- \`\` - перейти назад, в предыдущую папку перед прыжком.
- gG - lazygit
- go - открыть в duckdb (касается csv, xlsx файлов)
- e - скрыть-показать панель предпросмотра
- E - развернуть панель предпросмотра на полный экран
- R - открыть меню корзины
- T - открыть терминал для папки
- cz - создать zip-архив из выделенного
- c7 - создать 7zip-архив из выделенного
- cp - создать zip-архив c паролем
- M - меню монтирования и размонтирования блочных устройств (тех самых флешек и накопителей в том числе). Команды управления этим меню см. тут: [https://github.com/yazi-rs/plugins/tree/main/mount.yazi]

### Какие улучшения?

#### Niri

- Незначительно скорректированы шорткаты, добавлены Win+Shift+X и Win+Shift+C для засовывания окна в колонку и вытаскивания его из неё.
- Добавлен вышеупомянутый шорткат для вызова Neovide, так как с текстами и кодом нужно работать постоянно.
- Нажатие на кнопку выключения обрабатывается выводом меню, а не погружением в сон или выключением (но кое-где требуется явно установить `HandlePowerKey=ignore` в `/etc/systemd/logind.conf`).
- Закрытие крышки ноутбука вызывает событие noctalia lock-and-suspend, а не просто суспендит.

#### Yazi

В этом сетапе он сделан основным менеджером файлов, вызывается по Win+E (но у вас всегда будет под рукой и старый-добрый Nautilus, который идёт по умолчанию с CachyOS)

- Архивы открываются в отдельном окне в отдельной копии yazi, они монтируются во временную папку. Если архив слишком большой - используется ratarmount, если поменьше - fuse-archive с отключённым кешированием, если обычного размера, то просто fuse-archive. Вы не сможете закидывать в эти архивы новые файлы, зато сможете копировать из них то, что в них уже лежит.
- Добавлены плагины предпросмотра для csv, xlsx, json, md

#### Kitty

Добавлена обработка кириллических букв для базовых действий (копипаста, останов, выход). Шрифт сделан покрупнее, настроена прозрачность.

#### Neovim

Настроен масштаб Neovide как 0.9 (из коробки он выводит шрифт слишком крупно). Настроена работа с буфером обмена (помните, в режиме вставки буфер доступен по Ctrl+R). Добавлены плагины:

- csvview для красивого вывода таблиц (по умолчанию выводит таблицы красиво, но можно отключить/включить принудительно командами "CsvViewEnable", "CsvViewDisable", "CsvViewToggle")
- mini.surround, позволяющий добавить кавычки или любой иной символ вокруг выделенного текста (gza+символ), заменить окружающие текст символы на иной (gzr+старый символ+новый символ), а также удалить окружающие текст символы (gzd+символ)
- nvim-spider для того, чтобы ходить по частям названий в CamelCase или snake_case

#### Fish

- По нажатию Alt+e можно открыть nvim и править команду прямо в нём.
- Прямо в строке ввода можно переключаться на Normal Mode из vim и выходить из него с помощью i,a,I,A. Это расширяет возможности навигации по длинным командам.
- Актвирован zoxide.

### Зачем вообще это нужно?

Из коробки CachyOS c Niri cовершенно не раскрывает потенциал системы, мне захотелось сделать решение, которое бы ощущалось лишь чуть менее проработанным, чем KDE, но с гиковским духом. Поэтому в итоговом результате мы должны получить неплохо настроенный менеджер Yazi, Neovide, Noctalia. А в настройки Niri также добавлен сетап, который раскрывает её возможности по перемещению окон. В качестве просмотрщика картинок используется IMV, для чтения pdf/epub/fb2/djvu используется Zathura, установлен Libre Office и настроен файл mimeapps, чтобы не возникало проблем при открывании файлов.
